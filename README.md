# Python Mini Projects

A collection of Python mini-projects to help you learn and explore different programming concepts and techniques. These projects vary in difficulty and cover a wide range of topics including algorithms, data structures, web scraping, automation, and more.

## Projects

### Project 1: Calculator
A simple Python calculator that supports basic operations like addition, subtraction, multiplication, and division.

**Features:**
- Performs basic arithmetic operations.
- Handles division by zero errors.

### Project 2: To-Do List
A Python application that lets users create, update, and delete tasks from a to-do list.

**Features:**
- Add tasks.
- Mark tasks as complete.
- Delete tasks.

### Project 3: Web Scraper
A Python script that scrapes data from websites using the BeautifulSoup library.

**Features:**
- Scrapes articles, images, or any other data from websites.
- Saves the data to a CSV file.

### Project 4: Number Guessing Game
A simple number guessing game where the user has to guess the correct number within a limited number of attempts.

**Features:**
- Random number generation.
- Tracks attempts and displays success or failure messages.

### Project 5: Weather App
A Python application that fetches and displays the current weather for any city using an API.

**Features:**
- Fetches real-time weather data.
- Displays temperature, humidity, and weather description.

## How to Run

1. Clone this repository to your local machine:
    ```bash
    git clone https://github.com/your-username/python-mini-projects.git
    ```

2. Navigate to the project directory:
    ```bash
    cd python-mini-projects
    ```

3. Install required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the desired project:
    ```bash
    python project_name.py
    ```
